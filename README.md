# Fullstack Project

This repository is organized as a fullstack project:

- `backend/` — Flask backend application with PostgreSQL, JWT auth, and opportunity management.
- `frontend/` — frontend UI repository.

## Backend

See `backend/README.md` for setup, environment variables, API documentation, and testing instructions.

## Frontend

The `frontend` directory is reserved for the frontend UI implementation and should remain unchanged while backend improvements are applied.
